# **App Name**: EidSpark

## Core Features:

- Homepage & Navigation: Display a beautiful hero section with call-to-action buttons, responsive navigation bar, and links to core tools.
- Zakat Calculator: Provide a simple form for users to input various assets and instantly calculate their Zakat obligation.
- AI-Powered Eid Greeting Generator: Allow users to input a name, choose a template, and use an AI tool to generate a personalized Eid Mubarak card, with options to download or share.
- Salami Tracker & Leaderboard: Enable users to add people and amounts, automatically calculate the total 'salami' collected, and display contributions in a leaderboard-style UI. Requires user authentication.
- Eid Countdown Display: Show an animated, real-time countdown timer to Eid with an accompanying moon animation.
- Mosque Finder: Display nearby mosques using a map interface with markers for easy discovery.

## Style Guidelines:

- Primary color: Deep Emerald Green (#0A8033) to evoke elegance and cultural symbolism.
- Background color: Very soft, almost white, muted green (#EDF6EC) for a clean and calming aesthetic.
- Accent color: Vibrant Gold (#E9BE24) to add a festive and luxurious touch, providing contrast for calls-to-action.
- Headline and body text font: 'Inter', a grotesque-style sans-serif for a modern, objective, and neutral aesthetic fitting a startup feel.
- Use modern, clean line icons, complemented by custom illustrations of moon, lantern, and mosque for the hero section.
- Implement a fully responsive, mobile-first design with a sticky navigation bar for smooth user experience. Utilize rounded cards and soft shadows for a modern look, enhanced with subtle glassmorphism effects where appropriate.
- Incorporate smooth transitions, hover animations for interactive elements like feature cards, and an animated countdown for the Eid timer.
